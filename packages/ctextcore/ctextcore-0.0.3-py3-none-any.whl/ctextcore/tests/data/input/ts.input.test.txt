Xana i yini GEP?
Xindzuluti xa Mabindzu ya Gauteng - Gauteng Enterprise Propeller (GEP) i xiyenge lexi tumbuluxiweke hi Mfumo wa Xifundzankulu xa Gauteng, hi ku landza swipimelo swa Ndzawulo ya Nhluvukiso wa Ikhonomi, hi xirhumo xo nyika nseketelo na nhluvuko eka tiSMME eka Xifundzankulu xa Gauteng hi ku tirhisa timali na leswi nga riki mali.
Xana hikokwalaho ka yini ku tumbuluxiwile GEP?
