Yintoni i-GEP?
I-Gauteng Enterprise Propeller (i-GEP) (iLinge lesiQhubi laseGauteng) yiArhente eyasekwa nguRhulumente wePhondo laseGauteng, phantsi koququzelelo lweSebe lezoPhuhliso loQoqosho, ngomyalelo owodwa wokubonelela ii-SMME ngenkxaso nophuhliso kwiPhondo leGauteng ngamanyathelo ezemali nangengawo ezemali.
