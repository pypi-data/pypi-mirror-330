Iyini i- GEP?
IGauteng Enterprise Propeller (i-GEP) yi-Ejensi eyahlonywa mBuso wePhrovinsi yeGauteng, ngaphasi  kwesekelo lomNyango wokuThuthukisa zomNotho, ngokwegunya elibunqopha lokunikela ama-SMME isekelo kanye netuthuko ePhrovinsini yeGauteng Province ngokweemali kanye nalokho okungafaki iimali.
